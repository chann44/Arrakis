title: Configuration
description: All environment variables for configuring Arrakis services.
---

## Core

| Variable | Required | Description |
|---|---|---|
| `DATABASE_URL` | Yes | PostgreSQL connection string |
| `REDIS_URL` | Yes | Redis connection string |
| `SESSION_SECRET` | Yes | Secret key for session encryption |
| `API_PORT` | No | API server port (default: `8080`) |

## GitHub OAuth

| Variable | Required | Description |
|---|---|---|
| `GITHUB_CLIENT_ID` | Yes | GitHub OAuth App client ID |
| `GITHUB_CLIENT_SECRET` | Yes | GitHub OAuth App client secret |
| `GITHUB_REDIRECT_URI` | Yes | OAuth callback URL |

## GitHub App

| Variable | Required | Description |
|---|---|---|
| `GITHUB_APP_ID` | Yes | GitHub App numeric ID |
| `GITHUB_APP_PRIVATE_KEY_PATH` | Yes* | Path to PEM private key file |
| `GITHUB_APP_PRIVATE_KEY_BASE64` | Yes* | Base64-encoded PEM (alternative to path) |

*One of `PRIVATE_KEY_PATH` or `PRIVATE_KEY_BASE64` is required.

## Advisory sources

| Variable | Required | Description |
|---|---|---|
| `NVD_API_KEY` | No | NVD API key (required if `nvd_enabled: true`) |

## AI Analyzer

| Variable | Required | Description |
|---|---|---|
| `AI_ANALYZER_ENABLED` | No | Enable AI analysis (default: `false`) |
| `AI_ANALYZER_MODE` | No | `docker` or `http` (default: `docker`) |
| `AI_ANALYZER_IMAGE` | No | Docker image (default: `arrakis-ai-analyzer:local`) |
| `AI_ANALYZER_TIMEOUT` | No | Container timeout in seconds (default: `180`) |
| `AI_ANALYZER_HTTP_URL` | No | URL for HTTP mode |
| `OPENAI_API_KEY` | No | OpenAI API key for AI analyzer |
| `OPENROUTER_API_KEY` | No | OpenRouter API key (alternative) |
| `AI_ANALYZER_MODEL` | No | Model name (defaults to Claude) |

## Logging / ClickHouse

| Variable | Required | Description |
|---|---|---|
| `CLICKHOUSE_HOST` | No | ClickHouse host |
| `CLICKHOUSE_PORT` | No | ClickHouse port (default: `9000`) |
| `CLICKHOUSE_USER` | No | ClickHouse username (default: `default`) |
| `CLICKHOUSE_PASSWORD` | No | ClickHouse password |
| `CLICKHOUSE_DATABASE` | No | ClickHouse database (default: `default`) |

Without ClickHouse configured, logs are written to stdout only.

## Docker / Traefik (self-hosted production)

| Variable | Required | Description |
|---|---|---|
| `TRAEFIK_ACME_EMAIL` | Yes | Email for Let's Encrypt certificate |
| `DOMAIN_RECORD_TYPE` | Yes | DNS record type (`A` or `CNAME`) |
| `DOMAIN_RECORD_VALUE` | Yes | Server IP or CNAME target |

## Image overrides

```bash
# Override published Docker images
TGE_BACKEND_IMAGE=your-registry/tge-backend:latest
TGE_WEB_IMAGE=your-registry/tge-web:latest
```

## Worker concurrency

The worker service uses hardcoded concurrency values:

| Queue | Concurrency |
|---|---|
| `scans` | 8 |
| `dependencies` | 10 |

To change these, modify `/apps/worker/main.go` and rebuild the image.

## Scheduler interval

The scheduler checks for due cron triggers every **30 seconds**. This is not configurable via environment variables — change the tick interval in `/apps/scheduler/main.go` if needed.

## Complete `.env.example`

```bash
# Core
DATABASE_URL=postgres://arrakis:arrakis@postgres:5432/arrakis
REDIS_URL=redis://redis:6379
SESSION_SECRET=change-me-in-production

# GitHub OAuth
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
GITHUB_REDIRECT_URI=https://yourdomain.com/v1/auth/github/callback

# GitHub App
GITHUB_APP_ID=
GITHUB_APP_PRIVATE_KEY_PATH=/run/secrets/github_app_key

# Advisory Sources
NVD_API_KEY=

# AI Analyzer
AI_ANALYZER_ENABLED=false
AI_ANALYZER_MODE=docker
AI_ANALYZER_IMAGE=arrakis-ai-analyzer:local
AI_ANALYZER_TIMEOUT=180
OPENAI_API_KEY=

# ClickHouse (optional)
CLICKHOUSE_HOST=
CLICKHOUSE_PORT=9000
CLICKHOUSE_USER=default
CLICKHOUSE_PASSWORD=
CLICKHOUSE_DATABASE=default

# Traefik (production only)
TRAEFIK_ACME_EMAIL=ops@example.com
DOMAIN_RECORD_TYPE=A
DOMAIN_RECORD_VALUE=
```
